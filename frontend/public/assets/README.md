# Resume Folder

Place your resume PDF file here and name it `resume.pdf`.

The resume will be accessible at: `/assets/resume.pdf`

**Instructions:**
1. Save your resume as `resume.pdf`
2. Copy it to this folder: `frontend/public/assets/`
3. The resume link in your portfolio will automatically work

**Note:** Make sure the file is named exactly `resume.pdf` (all lowercase).
